Scheduling Problems

  A common problem encountered with organizing a schedule is that the length of time to find an ideal, or even acceptable, solution increases exponentially with the size of the schedule. Schedules that fit this situation have the following elements, with the problem being the methods for assigning shifts and rating each combination of shifts assignments.

  •  Shift descriptions: times and days of week
  •  Staff list
  •  Rules for assignments that are either "acceptable" or "ideal"
  • Method for assigning shifts
  • Method for accepting or rejecting schedules
  •  Publishing format
  •  Distribution (email, etc.)

The automated scheduling process

1. Gather requirements documents

  • A staff list with the type and number of shifts that each MD should work
  • A calendar of the dates that each MD is unavailable
  • email documents to john@schedulesolve.com

2. Validation

  • Process email inputs
  • Create validation documents with colour coding and totals
  • Receive validation documents via email
  • Example Targets
  • Example Calendar
  • Accept or edit documents
  • Repeat or Solve

3. Solve

  • The solution is mostly an automatic process
  • Rule priority will be adjusted if the solution isn't ideal
  • Receive solution documents via email
  • Example Targets & Results
  • Example Calendar with Assignments and Rules
  • Outputs can be edited and used as the inputs for a modified solution
  • Repeat or Publish
